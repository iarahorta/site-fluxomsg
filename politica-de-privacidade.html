Política de Privacidade – FluxoMSG

1. Coleta de dados

Coletamos informações fornecidas pelo usuário, como nome, telefone e e-mail.

2. Uso das informações

Utilizamos os dados para contato, prestação de serviços e comunicação.

3. Compartilhamento

Não vendemos ou compartilhamos dados com terceiros, exceto quando necessário para execução do serviço.

4. Segurança

Adotamos medidas para proteger as informações.

5. Direitos do usuário

O usuário pode solicitar alteração ou exclusão dos dados.
